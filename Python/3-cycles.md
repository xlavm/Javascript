# Cycles

1. while

    ```PY
    dia = 0    
    semana = ['Lunes', 'Martes']
    while dia < 2:
        print("Hoy es " + semana[dia])
        dia += 1
    """
    output: 
    Hoy es Lunes
    Hoy es Martes
    """
    ```

2. for

    ```PY
    nums = [4, 78]
    for n in nums:
        print(n)
    """
    output:
    4
    78
    """
    ```
