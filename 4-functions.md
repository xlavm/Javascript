# Functions

1. Funciones de Flecha 

    Es recomendable usar funciones de flecha

    ```JS
    //funcion tradicional
    function sumar(a, b){
      return a + b 
    }
    //funcion de flecha
    const sumar = (a, b) => {
      return a + b
    }
    //funcion de flecha abreviada
    const sumar = (a, b) => a + b
    ```
